# TypechoMobile-plugin
Typecho Android 客户端的配套插件

## 使用
[下载最新版本](https://github.com/appdev/TypechoMobile/releases)，放在你的网站`plugins`目录下。

正常情况下你的目录应该是这样的：
```
··/usr/plugins/TypechoMobile
```

然后登录 Typecho 后台，启用插件即可。
